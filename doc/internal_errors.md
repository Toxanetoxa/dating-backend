# Коды для внутренних ошибок

документ будет постепенно обновляться и дополняться !

| Код  | Сообщение                   | Описание                                          |
|------|-----------------------------|---------------------------------------------------|
| 1000 | path not found              | по данному адресу нет русурсов                    |
| 1001 | resource does not exist     | ресурс не существует                              |
| 1002 | resource already exists     | ресурс уже существует                             |
| 1003 | invalid data                | ошибка в теле запроса / неверный формат запроса   |
| 1004 | validation error            | ошибка валидации запроса (неверно заполнены поля) |
| 1005 | invalid authentication data | неверные данные авторизации                       |
| 1006 | internal server error       | непредвиденная ошибка на сервере                  |
| 1007 | unauthorized                | требуется авторизация                             |
| 1008 | token expired               | токен истёк, нужно запросить новый                |
| 1009 | invalid hash                | неверный хэш (для номера)                         |
| 1010 | code already sent           | код уже был отправлен                             |
| 1011 | invalid verification code   | неверный код подтверждения                        |
| 1012 | user is blocked             | пользователь заблокирован                         |
| 1013 | not supported phone number  | телефонный номер не поддерживается                |
| 1014 | ip is blocked               | IP заблокирован изза большого кол-ва запросов     |
| 1015 | profile not completed       | профиль пользователя не заполнен. ошибка доступа  | 
| 1016 | invalid file type           | формат файла не поддерживается                    | 
| 1017 | user geolocation not set    | не установлена геолокация пользователя            | 
| 1018 | have not match              | нет взаимного лайка. просмотр недоступен          | 
| 1019 | match already exists        | мэтч уже был создан                               | 
| 1020 | boost already exists        | буст уже был куплен                               |
